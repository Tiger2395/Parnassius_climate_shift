# Parnassius Climate Shifts: A Computational Analysis

**Author:** Tiger Wang
**Course:** Biology 152, University of Wisconsin-Madison
**Date:** Spring 2026

## Project Overview
This repository contains the R scripts, data pipelines, and harmonized datasets used for the independent research project: *"A Computational Analysis of Climate-Driven Elevational and Phenological Shifts in North American Parnassius Butterflies Using GBIF Occurrence Data."*

This study utilizes a computational data science approach to quantify the spatial and phenological responses of five North American *Parnassius* species (*P. behrii*, *P. clodius*, *P. eversmanni*, *P. phoebus*, and *P. smintheus*) to post-1990 anthropogenic climate warming.

## Repository Structure

* **`Parnassius_NA_Harmonized_Full.csv`**: The fully cleaned and harmonized dataset containing 7,188 curated occurrence records from the Global Biodiversity Information Facility (GBIF). Includes computationally extracted standardized elevations (via AWS terrain models) and transformed Day of Year (DOY) phenological variables.
* **`Elevation_analysis.R`**: The R script used to process spatial displacement. This file contains the Two-Sample t-tests used to evaluate significant elevational shifts between the pre-1990 and post-1990 temporal cohorts.
* **`Phenology_analysis.R`**: The R script used to evaluate temporal flight shifts. This file contains both the Two-Sample t-tests for analyzing continuous Day of Year (DOY) emergence and the Pearson's Chi-square tests for analyzing proportional changes in categorical seasonal emergence (Early vs. Late season).
* **`figures/`**: Contains the output visualizations generated by the R scripts:
    * `Parnassius_Elevation_Final_2Eras.png` (Elevational boxplots)
    * `Parnassius_Phenology_Final_2Eras.png` (Continuous DOY boxplots)
    * `Parnassius_Proportion_ErrorBars_2Eras.png` (Categorical emergence stacked bar charts)

## Data Sources
Raw biological occurrence data was aggregated from [GBIF](https://www.gbif.org/) and heavily relies on the contributions of community scientists via platforms like iNaturalist, as well as digitized historical specimens from global entomological collections (e.g., the Frost Entomological Museum, the Florida Museum of Natural History, and the Natural History Museum in London). 

## Requirements and Dependencies
To reproduce this analysis, you will need **R** (version 4.0.0 or higher) and the following R packages:
* `dplyr`
* `ggplot2`
* `stats`

## How to Run
1. Clone this repository to your local machine.
2. Ensure the working directory in R is set to the folder containing the `.csv` dataset.
3. Run the `analysis_scripts.R` file to generate the statistical summaries and output the data visualizations.
